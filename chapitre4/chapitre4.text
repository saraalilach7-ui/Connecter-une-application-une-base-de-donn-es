Dans ce chapitre nous avons appris à gérer les erreurs lors des opérations sur la base de données avec PDO on utilise try pour exécuter le code et catch pour intercepter les exceptions PDOException on peut activer PDO::ATTR_ERRMODE en mode exception pour que les erreurs lèvent des exceptions et ainsi les capturer on peut afficher les messages d’erreur de façon contrôlée ou les enregistrer dans un fichier log pour plus de sécurité et les logs sont utiles pour diagnostiquer les problèmes sans interrompre le programme
